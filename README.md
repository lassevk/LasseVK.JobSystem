# LasseVK.JobSystem

This project implements a set of NuGet packages for job scheduling and
processing for my .NET applications.

The goal is to have a system that can queue up jobs, process them, and handle
things like persistence, error handling, monitoring, and more.

**Note!** Most of the code in this project will be written by Claude AI, but
all of it will be reviewed and tested by me before it is released. Since it is
mainly a nuget package for my own use, I find this acceptable.
